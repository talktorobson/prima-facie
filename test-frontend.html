🚀 Fluxo Recomendado de Teste:

  1. Start Server: npm run dev para iniciar em localhost:3000
  2. Database Status: Verificar conectividade e dados em tempo real
  3. Admin Panel: Navegar pelo painel administrativo completo
  4. Subscription Management: Testar criação e edição de planos LaaS
  5. Case Billing: Calcular billing multi-modal com descontos
  6. Invoice System: Gerar faturas duplas (subscription + case)
  7. Time Tracking: Tracking de horas com billing automático
  8. Financial Management: AP/AR com workflow completo
  9. Client Portal: Acesso do cliente com self-service
  10. Reports & Export: Geração de relatórios Excel/PDF
